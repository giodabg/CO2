\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Introduzione}{Introduzione}}\label{index_intro_sec}
Il progetto "{}scontrini"{} implementa una pipeline locale\+:
\begin{DoxyItemize}
\item acquisizione immagine (foto scontrino)
\item pre-\/processing (Open\+CV)
\item OCR (Tesseract via pytesseract)
\item normalizzazione del testo
\item parsing euristico in strutture dati versionate (Receipt\+Contract\+V1)
\item persistenza su SQLite
\item esposizione API via Fast\+API (base per UI Web\+View su Android e Windows)
\end{DoxyItemize}\hypertarget{index_arch_sec}{}\doxysection{\texorpdfstring{Architettura (high level)}{Architettura (high level)}}\label{index_arch_sec}
